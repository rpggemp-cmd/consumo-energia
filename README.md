# ⚡ Consumo de Energia

## 📌 Descrição
Este projeto tem como objetivo calcular o consumo mensal de energia elétrica de um aparelho, com base na sua potência e tempo de uso diário.

## 🛠️ Tecnologias Utilizadas
- 🐍 Python
- 💻 Terminal / CLI
- 📊 Lógica de programação

## 🧮 Fórmula Utilizada
O cálculo do consumo mensal é feito utilizando a seguinte fórmula:

consumoMensal = (potencia * horasDia * 30) / 1000

Onde:
- Potência é medida em watts (W)
- Tempo de uso diário em horas
- O resultado é convertido para kWh

## 💰 Cálculo de Custo
Também é feito um cálculo estimado de custo mensal, considerando:

Valor do kWh = R$ 0,75
